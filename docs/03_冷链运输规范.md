# 冷链运输规范

## 1. 适用范围

本规范适用于生鲜食品、医药样本、低温原料等需要全程温控的物流订单，覆盖发货前预冷、干线运输、节点交接、末端配送和异常处置。

## 2. 温控要求

不同商品应按照业务规则设置温区。常见冷藏商品控制在 0 至 8 摄氏度，冷冻商品控制在零下 18 摄氏度及以下。运输前必须确认车辆、周转箱或冷柜处于稳定温区，禁止在未完成预冷的设备中直接装货。

每票冷链订单都应具备可追溯的温度记录，包括装车前温度、运输途中采样温度、到站温度和签收前温度。记录方式可以是温度记录仪、车载温控系统或人工点检表，但必须能回溯到订单或批次。

## 3. 运输与交接规范

1. 装车前检查商品包装完整性、保温耗材数量和温度记录设备状态。
2. 干线运输中按设定频次查看温控数据，发现偏离阈值应立即预警。
3. 到达分拨节点或仓库后，交接双方共同核对温度记录、封签和外包装。
4. 末端配送时优先安排早班派送，减少在常温环境中的暴露时长。

## 4. 温度异常处理

当监控系统或人工巡检发现温度超出阈值时，应立即执行以下动作：

1. 确认异常发生时间、所在节点和持续时长。
2. 检查制冷设备、电源、箱门密封或保温耗材是否异常。
3. 将异常订单标记为高风险，并通知值班运营和质量负责人。
4. 视商品属性决定是否隔离、复检或终止继续配送。
5. 补充记录处理动作、责任判断和客户通知情况。

## 5. 管理要求

冷链订单一旦出现温度异常，原则上应优先升级处理。涉及医药、婴幼儿食品或高价值生鲜商品的订单，不得仅凭经验判断继续流转，必须保留完整温控记录并走人工复核流程。
