# 运输费用计算规则

## 1. 适用范围

本规则适用于普通快运、冷链运输、同城急送和仓配一体业务的基础运费计算，主要用于报价说明和异常解释，不替代正式财务结算规则。

## 2. 基础计费逻辑

运输费用通常由首重费用、续重费用和附加服务费用组成。普通快运订单以实际重量或体积重较大者作为计费重量，冷链和特殊货物会在基础费用上叠加温控、保温或安全保障相关附加费。

常见计费要素包括：

- 始发地与目的地区域
- 商品重量或体积重
- 配送时效要求
- 是否属于特殊品类
- 是否需要上楼、签单返还或保温服务

## 3. 体积重规则

对于大件轻抛货物，应比较实际重量和体积重，取较大者计费。体积重通常按长、宽、高计算后除以约定系数得出，具体系数以承运商规则为准。客服在解释费用时应说明“按较大计费值收费”，避免只强调重量导致客户误解。

## 4. 特殊货物加价

以下场景通常存在附加费用：

1. 冷链商品：增加温控、冷媒和保温包装费用；
2. 易碎品：增加加固包装和操作保障费用；
3. 偏远地区：增加偏远地区派送附加费；
4. 超长超重件：增加特殊装卸与运输资源费用。

## 5. 客服解释口径

当客户对运费提出疑问时，应优先核对订单的重量、体积、路线和附加服务项，再说明费用构成。没有核对数据前，不要直接承诺“系统算错了”或擅自减免。
