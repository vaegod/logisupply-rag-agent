# 客户投诉处理规则

## 1. 适用范围

本规则适用于客户对物流时效、服务态度、商品破损、签收异常、售后处理不及时等问题发起的投诉处理场景。

## 2. 投诉分级

为便于响应和升级，投诉按影响程度分为三级：

1. 一级投诉：一般咨询或轻度不满，不涉及赔付和舆情风险。
2. 二级投诉：明确表达不满，要求补偿、加急或升级处理。
3. 三级投诉：涉及高价值订单、冷链医药品、平台舆情、重复投诉或监管风险。

## 3. 响应时效

- 一级投诉：30 分钟内首次响应，当日闭环。
- 二级投诉：15 分钟内首次响应，4 小时内给出处理方案。
- 三级投诉：10 分钟内升级主管，并持续跟踪直到闭环。

首次响应至少包括问题确认、当前处理状态和下一次反馈时间，不能只回复“已收到，请等待”。

## 4. 标准处理流程

1. 核验订单号、投诉内容、联系方式和历史沟通记录。
2. 判断属于时效、破损、服务态度还是售后争议。
3. 同步查询物流轨迹、异常工单和责任节点。
4. 给出明确的处理路径，例如补发、赔付、升级或再次回访。
5. 在客服系统中记录处理结果和客户反馈。

## 5. 升级要求

如果客户连续投诉两次以上、在社交平台公开发声、涉及特殊品类或明确要求管理层介入，应立即升级至客服主管或运营负责人。升级后仍需由首位接待人员持续跟踪，不得出现信息断层。
