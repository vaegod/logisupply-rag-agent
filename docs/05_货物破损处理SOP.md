# 货物破损处理SOP

## 1. 适用范围

本 SOP 适用于仓储、运输、分拨、末端配送等环节发现的货物破损场景，尤其适用于电子产品、玻璃制品、家居用品和其他易损商品。

## 2. 破损判定原则

货物破损包括外包装明显破裂、箱体受潮变形、封条缺失、商品本体破裂或无法正常使用等情况。若仅为轻微压痕但不影响商品本体，应在交接记录中注明并继续观察；若已影响商品安全性、完整性或客户体验，则应立即转入破损处理流程。

## 3. 处理流程

1. 第一发现人拍摄外包装、箱唛、面单和破损部位照片。
2. 将货物转移至异常区或指定隔离位置，防止继续流转。
3. 在系统中登记异常工单，补充订单号、节点、时间、责任班组和照片信息。
4. 由现场主管判断是否需要开箱复检，确认商品本体是否受损。
5. 结合包装责任、装卸记录和承运节点信息判断责任归属。
6. 根据规则决定补发、退回、赔付或客户签收备注方案。

## 4. 责任判定参考

- 入库前已破损：优先追溯供应商或上游发货责任。
- 仓内搬运破损：追溯仓储操作责任。
- 干线运输或分拨破损：结合交接照片、封签和装车记录判断承运责任。
- 末端派送破损：结合配送员签收前照片和客户反馈记录判断。

## 5. 客户沟通要求

客户投诉破损时，应先确认是否影响使用，再说明平台已启动异常处理。若需补发或赔付，应告知客户预计处理时效，并避免使用模糊措辞。对于高价值商品或客户情绪激烈的情况，应同步升级客服主管跟进。
