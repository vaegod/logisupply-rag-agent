# 退换货处理规则

## 1. 适用范围

本规则适用于平台自营和仓配一体业务中的退货、换货场景，覆盖普通商品、电子产品、生鲜冷链和活动订单。特殊品类如定制品、一次性拆封商品，应根据具体政策另行处理。

## 2. 退货条件

用户申请退货前应满足以下基础条件：

1. 在平台规定的售后时效内提出申请；
2. 商品主件、配件、赠品和发票信息齐全；
3. 商品不影响二次销售，或虽已开封但存在明确质量问题；
4. 不属于政策明确列出的不可退场景。

对于生鲜和冷链商品，如果因质量、变质或温控异常导致售后，一般需要附带照片、视频或签收时异常记录。

## 3. 换货流程

1. 客服核验订单、商品信息和售后原因。
2. 系统生成换货申请并通知仓库预留补发库存。
3. 原商品回收后进入质检环节，确认是否符合换货条件。
4. 审核通过后安排补发，并更新客户可查询的物流信息。

## 4. 退货入仓处理

退回商品到仓后应按照退货入仓标准执行：

- 外包装完好且商品状态正常的，进入可售质检流程；
- 存在损坏、污染或配件缺失的，进入异常品处理流程；
- 冷链商品须重点检查温控、保质期和签收异常记录。

## 5. 注意事项

客服回复时应明确说明售后时效、所需凭证和处理周期。若退换货原因涉及物流破损、延迟或温控异常，应同步关联物流异常工单，避免重复沟通和责任不清。
